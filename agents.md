# Market Intelligence Heat Map Generator

## Purpose
Executive healthcare intelligence platform focused on:
- leakage analysis
- ZIP geography
- referral pathway intelligence
- heat map intelligence
- CVSL operational strategy
- lead management systems analysis

## Architecture
analytics/ = deterministic intelligence engines
modules/ = Streamlit orchestration
renderers/ = executive infographic generation
utils/ = shared helpers and infrastructure

## Design Language
Philips-inspired enterprise healthcare aesthetic:
- Philips Blue (#0B5ED7)
- Dark Blue (#00126E)
- restrained accents
- executive readability
- clean hierarchy
- minimal clutter

## Rendering Philosophy
All renderers:
- use shared components.py
- generate 16:9 PNG exports
- optimize for executive PowerPoint use
- emphasize strategic interpretation over raw charts

## Analytics Philosophy
Python owns deterministic truth:
- healthcare semantic routing
- ZIP normalization
- referral pathway aggregation
- leakage scoring
- intensity classification

Avoid black-box calculations.

## Strategic Context
This platform is evolving toward:
- CVSL intelligence software

- operational healthcare strategy
- lead management systems intelligence
- executive healthcare rendering

## Coding Standards
- modular architecture only
- avoid monolithic files
- shared logic belongs in utils/
- renderer styling belongs in components.py
- avoid duplicated layout logic
