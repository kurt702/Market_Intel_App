# Market Intelligence Heat Map Generator

Executive healthcare market intelligence platform for:

- Leakage analysis
- ZIP geography intelligence
- Referral pathway analytics
- Heat map visualization
- Executive infographic rendering
- CVSL strategic analytics

---

# Purpose

This platform was designed to transform raw healthcare claims and referral datasets into executive-grade strategic intelligence outputs.

The system focuses on:

- Cardiovascular service line (CVSL) intelligence
- Referral retention
- Geographic market analysis
- Lead management operational strategy
- Network pathway optimization
- Executive infographic generation

The platform emphasizes deterministic analytics, semantic healthcare routing, and boardroom-ready visualization.

# Installation

## Clone / Download Project

Place project inside:

```text
Market_Intel_App/
```

## Install Requirements

```bash
pip install -r requirements.txt
```

## Run Streamlit App

```bash
python -m streamlit run app.py
```
